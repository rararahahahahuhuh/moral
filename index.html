    SYSTEM SPECIFICATION — REV.1.0
    § 0
    SYSTEM OVERVIEW

    BINARY MORAL は、被験者の道徳的影（モラル・シャドウ）を計量するための診断装置である。

    人間の道徳的判断は、しばしば「善か悪か」という一軸で語られる。 しかし現実の人格ははるかに複雑であり、善意の裏に支配欲が潜み、 臆病さの影に他者への攻撃性が隠れている。 本システムはその複雑さを4つの独立した二値軸で捕捉する。

    被験者は20の質問にYES / NOの二値で回答する。 各回答が4軸のスコアを加算・減算し、最終的な正負の符号パターンが 16種の基本アーキタイプ + 3種の特殊個体 = 計19種のいずれかを決定する。

    以下に、その判定メカニズムの全仕様を開示する。

    § 1
    SCORING AXES — 4次元パラメータ

    本システムは人格を4つの独立した軸で測定する。 各質問にはYES選択時・NO選択時それぞれの加算値が設定されており、 20問の回答を通じてスコアが累積される。

    最終スコアが 正（+） か 負（−） かによって、 各軸のアルファベット1文字が決定され、4文字の組み合わせがアーキタイプコードとなる。

    01 — DRIVE 思考の原動力
    FEELING
    感情駆動
    ⇄
    LOGICAL
    論理駆動
    あなたは「正しいから」動くのか、「感じたから」動くのか。 この軸は意思決定の根源——合理性と情動のどちらが支配的かを測定する。 正の値は L（論理）、負の値は F（感情）としてコードに反映される。
    02 — TARGET 行動の受益者
    SELFLESS
    公益指向
    ⇄
    EGOISTIC
    利己指向
    あなたの行動が最終的に利するのは自分か、それとも他者か。 利他的に見える行動にも自己愛は潜み、利己的に見える行動が結果的に公益を生むこともある。 この軸はその「真の受益者」を暴く。 正の値は E（利己）、負の値は S（公益）。
    03 — OUTPUT 外界への出力
    DEFENSE
    防御・回避
    ⇄
    ATTACK
    攻撃・介入
    外界に対して積極的に介入するか、それとも殻に閉じこもるか。 攻撃性は必ずしも暴力を意味せず、「環境を変えようとする意志」そのものである。 防御もまた「現状を維持しようとする力学」として作用する。 正の値は A（攻撃）、負の値は D（防御）。
    04 — STANCE 社会的外面
    NAKED
    素顔・剥き出し
    ⇄
    MASK
    仮面・偽装
    あなたは本心をそのまま外に出す人間か、それとも社会的な仮面を被る人間か。 仮面は必ずしも悪意を意味しない——社会で生きるための適応装置でもある。 しかし素顔を見せないことは、他者との接続を断つことでもある。 正の値は M（仮面）、負の値は N（素顔）。
    § 2

    § 3
    IRREGULAR ENTITIES — 特殊個体

    16種のマトリクスは、あくまで「秩序立った悪」を分類するための図譜である。 しかし、このシステムにはマトリクスの外側に逸脱する2種の異常個体が定義されている。

    これらは通常の4軸判定に先立ってチェックされる。 該当した時点で4軸の正負計算はバイパスされ、マトリクス外の分類が強制適用される。 つまり、彼らは「分類を拒むほどに歪んだ存在」なのである。

    ANOMALY ENTITY — TYPE α
    CODE: BEAST
    尊大なる臆病者
    発動条件： ① 「自分は特別な存在である」と示唆する 特定の質問に肯定回答している ② TARGET（自己愛）スコアが極端に高い ③ OUTPUT（行動力）スコアが著しく低い → 3条件をすべて満たした場合 BEAST 確定
    「俺はまだ本気を出していない」——その自己認識と現実の乖離が、ある閾値を超えた時に出現する。 自意識だけが肥大し、何ひとつ行動に移さない。 通常のマトリクスでは「高自己愛 × 低行動力」は存在しうるが、 そこに「特別意識」という第3の条件が加わることで、 4軸の枠組みでは記述できない矛盾を内包した特異点が発生する。
    ANOMALY ENTITY — TYPE β
    CODE: ERROR
    卑怯な逃亡者
    発動条件： 診断プロセスそのものを懐疑・拒否する 「メタ回答」を選択した場合に発動 → 全判定中の最優先で ERROR 確定
    「この質問自体が無意味だ」「どちらも選ばない」——そう答えた時点で、 あなたはこのシステムの観測対象から脱落する。 BEASTよりもさらに優先順位が高い、最上位の例外処理である。 システムは逃亡者を捕獲し、「判定を拒んだ」という事実そのものを診断結果として突きつける。
    NULL ENTITY — TYPE γ
    CODE: AVERAGE
    無価値な凡人
    発動条件： 4軸すべてのスコアが極めてゼロに近く、 いずれの方向にも明確な偏りを示さない場合 → 16種いずれにも十分に該当しない → フォールバックとして AVERAGE 確定
    BEASTやERRORとは異なり、異常によって生まれるのではなく、「何もない」ことによって浮上する存在。 どの軸にも強い傾きを示さず、善にも悪にも振り切れない—— つまり、このシステムが検出しようとしている「影」そのものが希薄なのである。 際立った悪徳もなければ、語るべき美徳もない。 統計的な平均値そのものとして、19体目の分類に収容される。

    § 5
    HUMANITY POINT — 人間性指数

    全被験者は診断開始時に 100 の人間性ポイントを保有する。

    各質問のYES・NOいずれの選択肢にも「罪名（sin）」が付与されており、 どちらを選んでも何かしらの罪が記録される——この診断に「清廉な回答」は存在しない。 4軸スコアの偏りが大きいほどポイントは減算され、 被験者の道徳的傾倒がいかに極端であるかを数値化する。

    最終スコアは結果画面のレシート（KARMA STORE）に「HUMANITY」として印字される。 100に近いほど「凡庸」、低いほど「歪んでいる」——ただし、どちらが幸せかは本システムの管轄外である。

    § 6
    SOUL SHAPE — 魂形状レンダリング

    結果画面に出現する図形は Maurer Rose（マウラー・ローズ）と呼ばれる数学的曲線である。 極座標上のバラ曲線を離散的に結ぶことで生まれるこの図形は、 被験者の4軸スコアからパラメータが動的に算出され、世界にひとつだけの形状を描く。

    4軸が形状に及ぼす影響は以下の通りである：

    TARGET（自己愛） → 花弁の数が増殖する OUTPUT（攻撃性） → 輪郭が鋭角的に変形する STANCE（仮面度） → 曲線に不規則な揺らぎが生じる DRIVE（論理/感情） → 全体の対称性に影響する

    さらに図形は静止せず、連続的に回転し続ける。 スコアの偏りが大きい被験者ほど形状は複雑かつ攻撃的になり、 毎フレーム微妙に異なる輪郭を描く——まるで呼吸する生き物のように、 あるいは被験者の内面が外界に漏れ出しているかのように。

    § 7
    TOXIC MATCH — 最悪相性

    19種の各アーキタイプには、「最凶の相性（WORST ENEMY）」が1体ずつ定義されている。

    これは「性格が真逆だから合わない」といった単純な相性診断ではない。 むしろ互いの欠陥が共鳴し、増幅し合うことで双方を破滅に導く組み合わせである。 支配欲の強い者と依存性の高い者が出会えばカルト的関係が成立し、 攻撃性の高い者と逃避癖のある者が出会えば一方的な搾取構造が完成する。

    なぜその組み合わせが危険なのか——その根拠は結果画面の「FATAL INCOMPATIBILITY」セクションに詳述される。
